(sort of like node modules and npm install if you're familiar with that)
Pipenv: 
https://realpython.com/pipenv-guide/ 
https://docs.python-guide.org/dev/virtualenvs/ 
https://pipenv-fork.readthedocs.io/en/latest/
https://pipenv.readthedocs.io/en/latest/advanced/#deploying-system-dependencies
https://github.com/pypa/pipenv/issues/3383

Either do pipenv install --system to locally install all the dependencies from the pipfile.lock or do pipenv run [usual python command here] to run the python command inside the virtual environment. pipenv install only installs to the virtual environment by default.
